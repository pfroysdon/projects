# Camera Gimbal

-----------------------------------------------------------------------------------

# Introduction
The "camera gimbal" is a full-functional autopilot and navigation system, built in C/C++ and uses a specifically designed PCB, that is able to control:
- 2-axis camera gimbal
- 3-axis camera gimbal
